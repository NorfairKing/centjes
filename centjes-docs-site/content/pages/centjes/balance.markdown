---
title: Balance report
description: Documentation about the centjes balance command, for viewing the balance of given accounts.
---


``` centjes balance
currency USD 0.01

account assets:cash
account equity:opening-balances
account expenses:coffee

2025-01-28
    | Opening balances
    * assets:cash 10 USD
    * equity:opening-balances -10 USD

2025-01-29
    | Coffee
    * assets:cash -5 USD
    * expenses:coffee 5 USD
```
